# FPV MultiBand Detector (1.2 / 3.3 / 5.8 GHz) на ESP32-S3

Три-діапазонний детектор FPV/дронів:

- 1.2 GHz – SM1370R / VM1373R (8 каналів)
- 3.3 GHz – FT3500 (8×8, 64 каналів)
- 5.8 GHz – RX5808 (48 каналів: A/B/E/F/R/L)

Функції:

- Режим **MONITOR** – ручний вибір діапазону/каналу, відображення RSSI.
- Режим **SCAN** – сканування поточного діапазону, пошук найсильнішого каналу.
- Режим **DETECTOR** – сканування ВСІХ трьох діапазонів, автоматичний вибір найсильнішого,
  перемикання відео на нього, тривога при перевищенні порогу.

MCU: **ESP32-S3** (DevKit)

Дисплей: **OLED 128x64 I2C (SSD1306)** через бібліотеку **U8g2**.

## Залежності

1. Arduino IDE:
   - Встановити **ESP32 core** (Board Manager → esp32 by Espressif Systems).
   - Обрати плату **ESP32-S3 Dev Module** (або свою S3-плату).

2. Бібліотека:
   - **U8g2** – через Library Manager:
     - `U8g2 by olikraus`

## Файли

- `FPV_MultiBand_Detector.ino` – основна прошивка.
- `FT3500.h` – драйвер FT3500 + таблиця частот.
- `Bands.h` – таблиці частот для 1.2 / 3.3 / 5.8 GHz.
